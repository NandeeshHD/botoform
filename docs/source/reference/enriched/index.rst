
enriched
########

Botoform abstracts and enriches the Boto3_ and Botocore_ projects.

The references in this document describe how we extend each project.

We use composition to provide many helper attributes and methods.

.. _Botoform: http://botoform.com
.. _Botocore: http://botocore.com
.. _Boto3: http://boto3.com
